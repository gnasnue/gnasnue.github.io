# 📦 PROJECT MANIFEST

이 매니페스트는 본 프로젝트의 핵심 철학과 구성 요소를 정의합니다. 모든 팀원은 기능 구현 시 아래의 원칙을 최우선으로 고려합니다.

---

## 🎯 1. Core Values (핵심 가치)
우리 프로젝트는 사용자의 삶에서 아래 세 가지 가치를 실현하는 것을 목표로 합니다.
1. **성장 (Growth):** 멈춰있지 않고 한 단계 더 나아가는 경험.
2. **방향 발견 (Direction Discovery):** 방황을 멈추고 나아갈 길을 명확히 함.
3. **목표 설정 (Goal Setting):** 추상적인 꿈을 실행 가능한 계획으로 구체화.

---

## 💡 2. Guiding Principles (운영 원칙)
* **의지력 배제:** 사용자의 의지에 의존하지 않고도 실행 가능한 **'시스템'**을 구축한다.
* **일상의 침투:** 사용자가 매일 접속하고 사용할 수 있는 매력을 제공한다.
* **즉각적 피드백:** 사용하기 쉬운 인터페이스를 지향하며, 작은 성취에도 즉각적인 보상을 준다.

---

## 📂 3. Documentation Inventory (문서 목록)
프로젝트의 세부 전략은 아래 문서들에서 확인할 수 있습니다.

| 파일명 | 용도 | 주요 내용 |
| :--- | :--- | :--- |
| `MANIFEST.md` | 프로젝트 지도 | 핵심 가치, 운영 원칙, 파일 구조 안내 |
| `WHYTREE.md` | 목적 분석 | 프로젝트 존재 이유 및 근본적인 목적 분석 |
| `PREMORTEM.md` | 리스크 관리 | 발생 가능한 문제점 사전 파악 및 대응 전략 |

---

> **Note:** "입력 안 하면 죽는다!" - 우리 서비스는 사용자의 성장을 돕는 가장 강력한 페이스메이커가 될 것입니다.
